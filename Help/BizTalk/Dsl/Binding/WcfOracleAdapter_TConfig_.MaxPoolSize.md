## WcfOracleAdapter<TConfig>.MaxPoolSize Property

Specifies the maximum number of connections in an ODP.NET connection pool.

```csharp
public int MaxPoolSize { get; set; }
```

#### Property Value
[System.Int32](https://docs.microsoft.com/en-us/dotnet/api/System.Int32 'System.Int32')

### Remarks

This is an ODP.NET property that is used for performance tuning. You must set MaxPoolSize judiciously. It is possible
to exhaust the number of connections available from ODP.NET, if this value is set too large.

The default is 100.
